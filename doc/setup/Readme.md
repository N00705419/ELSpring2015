## Hardware & Software

This will describe the setup regarding the pi and what's connected to it (hardware). It also will have information regarding the os, modules, sensors, and scripts.

- Raspberry Pi Model B
 - 16GB Class 10 SD Card
 - 40-pin GPIO cable
 - Cobbler
 - Temperature sensor
- Connections & Wi-Fi
 - SSH using Putty
- Raspbian OS & Tools
 - Python
 - nginx Web Server
 - sqlite3
